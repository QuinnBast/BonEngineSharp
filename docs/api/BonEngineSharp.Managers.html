Namespace BonEngineSharp.Managers

Classes

AssetsManager

Assets manager. Responsible to load or create game assets. Use this manager when you want to load textures, sound effects, music, config, ect.

CustomManager

A custom manager users can inherit from and register into the engine. If you need to implement your own custom manager, use this as a base class.

DiagnosticsManager

Diagnostics manager. Used to query diagnostics and debug data like draw calls count, FPS, ect.

GameManager

Game manager. Controls the general game flow / application. This includes exiting the application, changing scene, querying total delta time, ect.

GfxManager

Gfx manager. Responsible to rendering sprites, shapes, texts, and control the main window's properties.

IManager

Base interface for all managers.

InputManager

Input manager. Used to recieve user input, either by querying keyboard keys and mouse directly, or by binding keys to actions and querying action states.

LogManager

Log manager. Provides basic logging functionality.

ManagerGetters

Provide API to get all built in managers. Inherit from this class to get API similar to a Scene where you can just call Gfx.DrawImage(..) or Input.Down(..).

SfxManager

Sfx manager. Used to play sound effects and music.

UIManager

UI manager. Responsible to create, render and interact with UI systems.
